# Stone-Paper-Scissors Game (C++) 

## 📝 Description
This is a classic Stone-Paper-Scissors game developed as part of the **Programming Advices (Level 2)** curriculum. The project focuses on transitioning from basic syntax to structured programming and logical problem-solving.

## 🚀 Features
- **Interactive Gameplay:** Play against the computer in multiple rounds.
- **Random Logic:** Utilizes C++ random functions for computer moves.
- **Result Tracking:** Tracks wins, losses, and draws for each session.
- **Clean Code:** Structured using functions and clear naming conventions (Divide & Conquer).

## 🛠️ Technologies Used
- **Language:** C++
- **Paradigm:** Procedural Programming
- **Concepts:** Loops, Conditional Statements, Functions, and Arrays.

## 📖 How to Run
1. Clone the repository.
2. Compile the `main.cpp` file using any C++ compiler (like GCC or Visual Studio).
3. Run the executable file.
